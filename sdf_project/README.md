# Auxetic + SDF Tools

This folder contains the standalone tools for SDF/auxetic experiments.

## SDF tools
- `sdf_tools/sdf_3d.py`
- `sdf_tools/sdf_3d_gui.py`
- `sdf_tools/sdf_3d_pyvista.py`
- `sdf_tools/requirements-sdf.txt`
- `sdf_tools/requirements-sdf-pyvista.txt`

## Auxetic / lattice
- `rotating_squares_gui.py` (Tkinter DXF generator)
- `auxetic_gen.py` (re-entrant hex DXF)
- `lattice_gen.py` (square/triangle DXF lattice)

Copy this `sdf_project` folder to GitHub.
